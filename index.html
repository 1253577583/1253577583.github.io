计组存储器

发表于 2022-11-19

唐朔飞计算机组成原理，第四章存储器

4.1 概述

按存储介质分类，可分为半导体存储器、磁表面存储器、磁芯存储器、光盘存储器。
按存取方式分类，可分为随机存储器、只读存储器、顺序存储器、直接存储器。
按在计算机中的作用分离，可分为主存储器、辅助存储器、缓冲存储器。

存储器性能指标，速度、容量和每位价格。
缓存-主存层次主要解决CPU和主存速度不匹配问题。
主存-辅存层次主要解决存储系统的容量问题。

4.2主存储器

主存各存储单元的地址位置是由单元地址号来表示的。
地址总线用来指出存储单元的地址号。
计算机可按字寻址也可按字节寻址。

存储容量指主存存放二进制代码的总位数。
存储容量 = 存储单元个数 X 存储字长

存储速度由存取时间和存取周期表示。
存取时间分为读出时间和写入时间。
存取周期指存储器进行连续两次独立存储器操作的最小间隔时间。

存储器带宽，表示单位时间内存储器存取的信息量。

线选法，用一根字选择线，直接选中一个存储单元的各位。
重合法，XY两方向的选择线，选择任意一位。

集中刷新，在规定的刷新周期内，对全部存储单元集中刷新。
分散刷新，对每行存储单元的刷新分散到每个存取周期完成。
异步刷新，每隔一段时间刷新一行，在刷新周期内完成刷新。

位拓展，指增加存储字长。
字拓展，指增加存储器字数量。
字位拓展，既增加存储器字数量，也增加存储字长。

CPU低位地址线与存储器芯片的地址线相连。
CPU高位地址线，或用于扩充，或用于片选。

必须对存储器芯片进行扩位，使其数据位数和CPU数据线数相等。

CPU读写命令线一般直接与存储器芯片读写控制端相连。

片选线，通过逻辑电路，产生有效的选择信号。

系统程序、标准子程序、常数使用ROM。

汉明码，编码距离越大，纠错检错能力越大。
添加检测位，分析奇偶校验，利用检测位进行纠错定位。

4.3 高速缓冲存储器

程序访问的局部性，将近期要用到的程序或数据，提前从主存存入cache。
主存缓存都分为若干块，主存缓存块大小相同。
缓存地址分为块号和块内地址，主存地址也分为块号和块内地址。

平均访问时间使用cache命中率，对访问主存时间和访问cache时间进行加权平均。
访问效率，访问cache时间 / 平均访问时间，表示cache的作用效果。

增大块长，由与局部性使命中率有提高。
过大的块长，会使缓存装入的有用数据少于被替换的有用数据。

写直达，写操作既写入cache，又写入主存。
写回法，写操作只写入cache，在cache替换时，再写回主存。
使用标志位来表示cache数据是否与主存一致。

计组系统总线

发表于 2022-11-18

唐朔飞计算机组成原理，第三章系统总线。

3.1 总线的基本概念

总线是共享的传输介质，由许多传输线组成，可串行或并行传输。

3.2 总线的分类

片内总线，也就是CPU内部，寄存器之间的总线。

系统总线，CPU与主存，IO设备等大部件之间信息传输线。

系统总线又可以分成3类，数据总线，地址总线和控制总线。
数据总线，传递数据信息，是双向传输总线。
地址总线，用于指示数据总线上，源数据或目的数据，在主存单元或IO设备的地址。
地址总线上的信息由CPU单向传输，地址线位数与存储单元个数有关。
地址线和数据线是所有部件共享的，决定当前由哪个部件使用总线，需要控制信息。
控制总线，用来传输控制信号的线路，既有CPU发出的控制信号，也有部件发出的中断请求。

通信总线，用于计算机系统之间，或与仪表盘之类的通信。
串行指一位一位地，按顺序分时传送。
并行指在多条并行的线路上，同时由源传送到目的地。

3.3 总线特性和性能指标

机械特性，机械连接方面的性能，几何尺寸，形状，插头插座的标准。
电气特性，信号的电平范围，是高电平有效还是低电平有效。
功能特性，地址总线用来指出地址码，数据总线用来传递数据，控制总线发出控制信号。
时间特性，总线中的任一根线在什么时候有效，信号互相存在时序关系。

总线宽度，通常指数据总线的根数，8位、16位、32位、64位。
总线带宽，总线的数据传输率，每秒传输的字节数，工作频率 X 总线宽度。

总线标准，解决系统、模块、设备之间的适配问题。

3.4 总线结构

单总线结构，所有的传送通过一组总线，很容易出现传输瓶颈。
双总线结构，将速度较低的IO设备分离出来，使用通道和IO总线连接。
三总线结构，包括IO总线，主存总线，DMA总线。
三总线结构，包括局部总线，系统总线，拓展总线。
四总线结构，包括局部总线，系统总线，高速总线，拓展总线。

PCI总线，将高速、高性能的外设，尽量靠近CPU总线。
通过PCI桥与CPU总线相连，驱动能力不足可采用多层结构。（zz不回答我的疑问）

3.5 总线控制

总线上的设备，依据有无控制总线的能力，划分为主设备和从设备。
多个主设备同时要使用总线时，就需要总线判优控制。

链式查询，BS总线忙、BR总线请求、BG总线同意。
计时器定时查询，从0开始计数，或循环计数，增加设备地址线，设备地址与计数值一致，则获得总线使用权。
独立请求方式，每台设备都有BR、BG，总线控制部件进行排队。

总线通信控制，完成一次总线操作被称为总线周期。

申请分配阶段，主设备提出申请，经仲裁机构决定下一传输周期的使用权。
寻址阶段，主设备发送地址和有关命令，启动参与本次操作的从模块。
传数阶段，主设备和从设备进行数据交换，数据从源发出，流入目的模块。
结束阶段，主设备信息从总线上撤出，让出总线使用权。

习题指导

USB是串行总线，ISA是系统总线，PCIE是串行传输。
中断信号通过数据线传输，突发传输方式需要一次地址时间。
注意双向传输带宽要乘2，注意题目给的公式的具体细节。
局部总线，指在系统外为两个以上模块提供高速传输信息通道。
突发工作方式，被传送的数据在主存中连续存放，访问时只需给出第一个数据的地址。
波特率指单位时间传输的位数，比特率指单位时间传输的有效位数。

笔记本更换矛盾

发表于 2022-10-09

自2020年9月份首次选购笔记本至今，随着笔记本的需求和使用感受越发清晰。
此间存在的矛盾逐渐显现，笔记本更换似乎成为可以考量的事情。
但对于更换高性能笔记本（以下简称游戏本），仍然在许多方面存在各样的矛盾。

主要需求因素

于我而言，其实游戏本主要用来解决游戏需求，也就是玩。
如果说游戏本用来玩是奢饰品属性的话，好像游戏本也是各方面的性能提升，
确实会在某些非游戏方面带来更好的使用体验。
如果说游戏本是必需品呢，似乎又有些言过其实，
毕竟当下的笔记本可以满足学习方面的使用需求。
简而言之，就是游戏需求同合理性的考量。

主要心理因素

就大学生活而言，娱乐的需求出现是比较合理的。
但现有的设施只能提供狭隘的娱乐体验。
具体可以划分为，手机游戏，娱乐性电子信息，现实娱乐。

电子游戏显而易见就是运行在手机上的游戏，
但常常由于手机性能限制，相较于电脑端游戏体验性差距很大。

娱乐性的电子信息，常见的是网络小说和短视频，以及各类看似传递实用信息，实质为碎片垃圾的生活指南和实事解读。
前者问题在于数量庞大良莠不齐，遇见优秀作品的概率较低，平均下来体验自然不好。
后者问题在于信息过于零散，而且来源没有专业性和可靠性，看似学习知识的伪装，带来的是时间上的浪费。
同时信息类娱乐方式很大的特点是带来大脑的过度疲劳，
从实际体验来看，大脑不会区分有用和无用的信息，它只能被动的处理，
这时倘若接收的信息过多，不仅会带来大脑的负担，同时也会降低对有效信息的理解能力。
所以，无论如何尽量避免选择这种娱乐性电子信息作为娱乐方式。

现实娱乐，在疫情当下，封校常态，每日核酸的时代，难能可贵。
既不存在与大自然亲密接触感悟世界的可能，也不存在漫步市井遍寻人间烟火的机会。
至于人与人交往的火花，在教育背景和学业状况下(后续文章会专门分析)鲜有可能。
到这个地步，除了体育活动外，很难有称得上的比较像样的娱乐活动。

家庭经济因素

其实这里差不多就是核心矛盾了，也是问题之所在。
从家庭环境来说，谈不上贫困，但也说不上富裕，
收入较为固定的情形下，为了保持家庭的良性发展，积累财富的主要方式是节俭。
这种量入为出的方式，其实会导致一方消耗增多，另一方就会降低生活体验的困境。
这种时候常常采用的就是折中，也就是妥协的方式达到平衡。
在不必要的支出上，娱乐支出首当其冲，
减少娱乐既能降低花费，也能将富于时间用于其他有益的事物上去。
于此伴随而来的自然就是生活的压力和对环境的不满，
这种情绪不可避免的产生，倘若认识不到也许会带来很大的损失。

实际情形的自我体验

当局者迷，旁观者清。
来到千里外的北国，在冰雪洗礼下自然也会产生不一样的认知。
感受孤独，体会自我的本质，才越发能看见那些抽象的呈现。

那杂乱无章的堆放，不仅是空间的堵塞，
其实也是压力的呈现，寓示着无法放下无法抛弃的生活重担。
看似每次对于事物意见的分歧，其实也是一种知识上匮乏，
以及对自我的表达，也是压力的释放方式。

身在其中，难免会出现问题而不自知，
也会出现自我压抑但得不到理解很支持的孤独与苦闷。

缺失的始终是对生活的理解，以及真正知识的取得。
在当今时代，每个人依旧是孤独的个体，难以掩喻的孤独。。。

除了独立自主之外，别无获得自由取得幸福的途径，
国家如此，个体更是如此。

游戏、竞技和生活

发表于 2022-09-25

游戏能够带来什么，竞技运动的比重如何，究竟怎么生活？
视频流媒体代表了什么？
书籍与书籍，他们内在的区分是什么样的？

游戏与快感的结合

关键在于游戏和快感的关系问题。
游戏为什么会产生快感？
科学解释大概就是什么多巴胺啊、激素分泌之类。
然后以激素化合物种类来区分，休闲、上瘾等不同的表现。

其实部分中国人不习惯于这样理解问题。
更多的通过经验、感受来判断和解释各种现象。

但在科学文化的冲击下，我们逐渐放弃这种方式。
科学自有其力量和准确性，但独有的感觉感受才是人活着的核心。

倾听内心的声音，接收来自身体的反馈，也许能看到事物的另一面。
当我们去感受内心，感受自我，自然而然就会变得沉静，具有掌握力。

我个人感觉，上瘾这类问题突出在刺激性上。
刺激会带来力量也会带来疯狂，而倾听削弱了这样的刺激，让我们更能接触事情的内部。

部分游戏举例

火炬之光之类的单机游戏，快乐来源于等级的提升、装备的获取、随机性地图和剧情的展开。
这其中蕴藏着，扮演角色自身能力的增强，但往往到达了顶点，游戏兴趣会快速下滑。

比如说奥日这种体验性游戏，大概是地图探索，收集、战斗、解密的结合。
无论内容多或少，最终会走向尽头，也就是全收集全成就。

再比如植物大战僵尸，在解锁全部植物后，打败僵尸博士，大概也就到了尽头。
那植物大战僵尸里的无尽模式怎么理解呢？
俄罗斯方块就是很好的例子，俄罗斯方块为什么有人去玩。
更有甚者，为了突破记录乐此不疲，单调的游戏不会使人厌倦吗？

这时就引出另一个关键词，竞技。

竞技活动是发展的起源

竞技，显而易见是和其他人的比较斗争活动。
但事实并非如此，竞技永远只是自我的突破，个人的提升。

支撑俄罗斯方块的源头，是突破记录的欲望。
突破记录的背后，实质上是对自我反应力和耐心等能力的锻炼。
更实际的是，对自我身体思维精确控制力带来的愉悦。

支撑体育竞技运动的，是内心对极限的渴望。
锻炼，其实就是对自身活力状态的磨砺。
意识不到这点，而出于压迫进行过度锻炼的运动员，中将面临健康状态恶化的处境。

得到的欲望和正确的认知塑造出强大的个体。
大方的态度和真诚的态度建立起和谐的关系。

生活的行为

每天我们都要做各种事，每一天我们都要做各种事。
我们学习到经验，体会到教训，感受到流逝。

但每个人都是孤独的个体，无论家庭、亲友多么亲密无间。
只有去发掘自己的内心，才有可能窥探出世界的一隅。
自己的感受和思维是我们唯一可以信赖的初始条件，而行为则是我们给出的解答。

只有体会孤独，才不会害怕孤独。
只有深入了解孤独，才能将其视为一种存在，而不是一种阻碍。

现在的视频流媒体恰好应和了这种孤独，可以理解为野蛮的占有人的时间。
对话形式的讲述方式，图像化的表达，让我们误以为这是和人在对话。
这种随时随地的对话方式，使我们不再面对自我，同时也让我们忘记自我。

以探索世界，获取知识，得到体验的眼光看待流媒体，我们才能够得到我们需要的和我们想要的。
人类活动实践，在于得到，在于提升，这时强大欲望塑造的探索能力，使人站到的高峰。

总有一些我们认为正确的、重要的事情，也有不正确、不重要的事情。
但人做事从来不是以正不正确、重不重要为依据，而是以我想不想做来判断。

贴合内心的想法，感受行为的每个瞬间，品味那独特的当下。
我们能感受到事物内在的本原，体会世界的流动。
这个时候，生活的本能会指引我们走向何方，生命是精彩的绚烂的绽放。

书籍之间的距离

从小说到专业书，也是从虚幻到现实的过程。
阅读小说，是想象力的挥洒的空间，体会到思维内在的满足。
专业教材，是现实现象和理论的呈现，是把握现实改变世界的力量。

想象与现实交织，带来更多现实与想象边界的问题？
谁能肯定这不是一场梦，谁又能肯定虚幻不是另一种现实？

我们无法决定我们是什么，我们只能考虑去做什么。
我们别无依靠，姑且相信的只有我们自己的内心，自我的感受。

不要去想，要去感受。

识字问题初步分析

发表于 2022-09-24 更新于 2022-09-25 分类于 brother

显而易见，不识字的问题是和同龄人比较而得知的。
从这个短语，看不出问题的程度，建议问题还是表述清晰一些比较好。

问题可能来源？

1.首先考虑到的是幼儿园阶段的教育：
如果是年龄段相同，为什么小宝识字能力弱。
如果是年龄段不同，大概就是差了一年的学习时间。

2.其次是家庭教育中的问题：
既然已经做了认字的准备工作，为什么结果不是很好。
如果父母态度很随意，那结果基本上显而易见。

3.身为不识字小宝本身的问题：
是没有识字的兴趣，对识字很抵触。
还是在识字中遇到困难，但又无从解决。
还是年纪太小，对于主动学习没有什么概念。

可能解决方案考虑？

处于对小宝教育环境的陌生，和陪伴时间较短，无从给出针对性建议。
但对于我所知的情形，依旧有些行为可以改善。

对于小宝这个年龄段的儿童，环境比较重要，父母的言行和家庭环境有很大影响。
家庭的整洁度，作息规律程度，户外活动频率，父母的情感表达等。。

  • 首先可以考虑逐步递进，每天可以睡前认字10或20个，重在坚持。
  • 其次对可能遇到的课程压力，也许需要一些额外的努力。
  • 身为父母，必须以身作则，至少在识字期间要陪伴关注他。
  • 关注户外活动，抽出时间散步，于小宝交流，了解他的想法。

尽量避免的问题

依我之见，家中的整洁程度堪忧，空间被不必要的杂物占据。
也许，思考和想象的空间也随着这样的环境被夺走了。

对儿童，我的理解是陪伴是最好的鼓励。
要去陪着他识字，不要嫌简单麻烦或者进度慢，更别提看短视频小说了。

趣味性的激励，抓住他的欲望和兴趣。
比如以故事的形式教他识字，或者对识字后前景的展望。

看似问题是识字问题，其实可能根源不在这里。
如果父母本身理不清生活的秩序和趣味，小宝也会受其影响。
